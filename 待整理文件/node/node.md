### 网络

#### path 模块

* extname(path) 返回后缀名
* dirname(path) 返回当前文件的目录
* parse(path) 返回解析后的路径

```
path.parse('/home/user/dir/file.txt');
// 返回:
// { root: '/',
//   dir: '/home/user/dir',
//   base: 'file.txt',
//   ext: '.txt',
//   name: 'file' }
```

> 统一资源定位符

* url的构成

* url 解析

#### http 协议

> 超文本传输协议
> 规定了网页传输内容的规则
> 规定了数据在浏览器与服务器之间传输数据的过程和格式。

* 细节
  * 应答形式的协议，一次请求对应一次响应。

#### 软件开发模式
  * 单机软件
  * C／S软件
    * 客户端 Client ／ 服务器端 Server
    * 需要下载安装之后才能使用
  * B／S软件
    * 浏览器 Brower ／ 服务器 Server
    * 打开网址之后才能就可使用
  
  * DNS 服务器解析域名 返回 IP 地址

  * 测试服务器是否可以跑通
    * ping 网址

#### 网络爬虫

  * 按照一定的规则，自动地抓取万维网信息的程序或者脚本。

  * 开发思路：
    * 打开网页查看内容，找图片地址（找规律）
    * 获取所有html内容
    * 通过正则表达式提取所需内容
    * 遍历拿到的内容，一个一个请求
    * 最后将获取到的数据保存在本地
